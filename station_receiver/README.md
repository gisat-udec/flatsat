# Flatsat

Sketch de Arduino para la Raspberry Pi Pico W que recibe datos por wifi y los retransmite a un sitio web con WebUSB

### Dependencias

- Arduino IDE 2.3.3
- Arduino-Pico 4.2.1 https://github.com/earlephilhower/arduino-pico

### Hardware

- Raspberry Pi Pico W
